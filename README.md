# Personal Portfolio (GitHub Pages)

This is a one-page personal portfolio website ready for GitHub Pages.

## Customize

Edit these files:

- `index.html`
  - Replace `Your Name`, email, LinkedIn, and GitHub links.
  - Update project titles, descriptions, and URLs.
- `styles.css`
  - Change colors in `:root` if you want a different theme.
- `script.js`
  - Keeps scroll reveal animation + auto year in footer.

## Publish using GitHub web

1. Create a new GitHub repository.
2. Open the repository in your browser.
3. Click **Add file** -> **Upload files**.
4. Upload `index.html`, `styles.css`, `script.js`, and `README.md`.
5. Commit the files to `main`.
6. Go to **Settings** -> **Pages**.
7. In **Build and deployment**, set:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` and `/ (root)`
8. Click **Save**.
9. Wait 1 to 3 minutes and open the URL shown on the Pages screen.

If your repo is named `username.github.io`, your site URL is:

`https://username.github.io`

Otherwise:

`https://username.github.io/repository-name`
